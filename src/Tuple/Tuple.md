## タプル
複数の違う型の要素を格納して1つの値にしたり、固定長リストとして使用する
```haskell
(1,'a'')

(1,'a',"AAA")

[(1,2), (1,2,3)] -- 形が違うので怒られる
--  Couldn't match expected type: (a, b)
--              with actual type: (a0, b0, c0)

fst (1,2) -- ペアのみ
-- 1

snd (1,2) -- ペアのみ
2

zip [1,2,3,4,5] ["A", "B", "C", "D", "E"]
-- [(1,"A"),(2,"B"),(3,"C"),(4,"D"),(5,"E")]
zip [1,2,3,4,5] ["A", "B", "C"] -- 余りは無視される
-- [(1,"A"),(2,"B"),(3,"C")]
```
